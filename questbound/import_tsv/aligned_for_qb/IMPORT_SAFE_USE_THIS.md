# Import Safe Files (Use These)

Use these files in Quest Bound import dialogs:

- Attributes -> `attributes_with_options.tsv`
- Actions -> `actions_import_safe.tsv`
- Items -> `items_import_safe.tsv`
- Charts -> `charts_import_safe.tsv`
- Documents -> `documents_import_safe.tsv`
- Archetypes -> `archetypes_import_safe.tsv`
- Statuses -> `statuses_import_safe.tsv`
- Keywords -> `keywords_import_safe.tsv`

Notes:

- `attributes_with_options.tsv` is required for list attributes because QB validates `options` arrays.
- Scripts are still manual in most QB builds; use `../scripts.tsv` as implementation reference.
